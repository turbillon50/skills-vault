## Qué es V-Gift

V-Gift es una plataforma donde puedes regalar lo que sea, a quien sea, sin complicarte. Un solo lugar para encontrar el regalo perfecto — desde gift cards y recargas telefónicas hasta vuelos, experiencias turísticas y productos físicos.

## El problema que resuelve

- Se te olvidan cumpleaños, aniversarios, fechas importantes
- Quieres regalar algo pero no sabes qué
- Tienes que brincar entre 5 apps distintas para comparar opciones
- Regalar desde México al extranjero (o viceversa) es un dolor de cabeza

## Cómo funciona

1. **Conecta tus contactos** — Sincroniza tu agenda y registra fechas importantes (cumpleaños, aniversarios, días especiales)
2. **Recibe recordatorios** — V-Gift te avisa antes de que se pase una fecha, con sugerencias personalizadas según el perfil del destinatario
3. **Elige del catálogo** — Un catálogo amplio y diverso: gift cards de marcas globales y locales, recargas de celular, boletos de avión, tours y experiencias, productos curados
4. **Regala en un click** — Paga, personaliza el mensaje y el regalo llega directo al destinatario (digital al instante, físico con tracking)

## Para quién es

- **Cualquier persona** que quiera regalar sin estrés
- **Empresas** que quieran programar regalos corporativos (clientes, empleados, partners)
- **Familias** distribuidas geográficamente que quieren mantenerse conectadas con detalles

## Catálogo (proveedores integrados)

| Categoría | Proveedor | Alcance |
|---|---|---|
| Gift Cards | Reloadly | 150+ países, miles de marcas |
| Recargas telefónicas | Reloadly Topups | 800+ operadores mundiales |
| Vuelos | Duffel | Aerolíneas globales |
| Tours y experiencias | Viator | 300k+ actividades en 200+ países |
| Pagos de servicios | Reloadly Utilities | Servicios básicos |

## Diferenciadores

- **Recordatorios inteligentes** — No se te pasa ninguna fecha
- **Catálogo universal** — Digital + físico + experiencias en un solo lugar
- **Social por naturaleza** — Conectado a tus contactos, no es un ecommerce frío
- **Multi-país** — Regala de México a Japón sin fricción
- **Personalización** — Mensaje, presentación y sugerencias basadas en gustos del destinatario

## Stack técnico

| Capa | Tecnología |
|---|---|

| Auth | Clerk (role-based) |
| Base de datos | Neon Postgres |
| Pagos | Mercado Pago (MX) + Stripe (internacional) |
| Email transaccional | Resend |
| Gift Cards / Recargas | Reloadly API |
| Vuelos | Duffel API |
| Tours | Viator Partner API |
