# Professional Site Generator

**Trigger:** crea un sitio profesional, genera un sitio, pro-site

## Descripción
Generador de sitios profesionales con Next.js 14 + TypeScript + Tailwind + Framer Motion + Clerk.

## Stack
- Next.js 14 (App Router)
- TypeScript + Tailwind CSS
- Framer Motion (animations)
- Clerk (authentication)
- SEO optimizado
- Dark mode
- Mobile-first responsive
- Performance optimizado

## Uso
Di: crea un sitio profesional para [sector]
Yo genero todo listo para Vercel.
